# food_delivery_api

#### Clone The repo

`$git clone https://github.com/shofizone/foo_delevary_api.git`

#### Navigate to project dir

`$ cd foo_delevary_api`

#### Install dependency

`$ npm install`

#### Run The Project

`$ npm start`

### Configure Database

- This project depends on MongoDB.
- Default url `$ localhost:27017`
- To set custom DB url user environment variable `$ DB_URL`
